"Генератор лабиринтов"

Доступные команды в программе (регистр для ввода команд любой):
	1. new maze (очевидно)
		1.1. Создаёт лабиринт независимо от того, создан ли уже лабиринт (старый в любом случае затирается, если не сохранён)
		1.2. Ввод всех параметров построчный! Т.е. каждая координата записывается в новой строке
		1.3. При вводе координат входа и выхода они автоматически подстраиваются под сетку лабиринта, поэтому их значение может измениться
			на один (что есть толщина стенки)
		1.4. Один из параметров - способ генерации (вводится 'dfs' или 'mst'):
			DFS - обход в глубину
			MST - мин. остовное дерево

	2. solve
		2.1. Если создан или загружен лабиринт - решает его и отображает путь

	3. save
		3.1. Сохранение в файл, лежащий в папке с программой
	
	4. load
		4.1. Загружает файл, лежащий в папке с программой: он должен изменяться только через программу!
		4.2. Старый лабиринт, если есть, не сохраняется

	5. print
		5.1. Печатает текущий лабиринт

	6. exit
		6.1. Выход без сохранения текущего лабиринта
		6.2. Выдаст подтверждение, на которое ответом могут быть yes или no
